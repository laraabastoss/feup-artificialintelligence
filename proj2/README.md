# IA Project 2

Before running the program, the following commands should be executed:


```
pip install scikit-learn
```

```
pip install pandas
```

```
pip install numpy
```


```
pip install seaborn
```

```
pip install matplotlib
```

```
pip install imblearn
```


This will install all the necessary tools to run the project.

To run the code, open src.ipynb and simply click "Run all". You can also choose to run cell by cell.

All the documentation and explanation of the project is on the notebook file. The support power point was sent next to the project.
